# Candy ECE Crush Saga


## Description

Candy ECE Crush Saga est un jeu de puzzle développé en langage C, inspiré du célèbre Candy Crush. Le joueur doit aligner des bonbons identiques afin de marquer des points et progresser à travers différents niveaux de difficulté.
